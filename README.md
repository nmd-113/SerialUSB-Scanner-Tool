# SerialUSB Scanner Tool

**SerialUSB Scanner Tool** is a simple Windows app for reading data from USB serial and RS232 devices.

It is especially useful for **barcode scanners**, but it can also work with other serial devices that send data through a COM port.

---

## Main Features

* **COM port detection**  
  Finds available serial ports on your PC.

* **Easy port selection**  
  Choose the COM port used by your scanner or device.

* **Configurable baud rate**  
  Supports common baud rates from **4800** to **115200**.

* **Auto-connect on startup**  
  Can automatically connect using the saved settings.

* **System tray support**  
  The app can stay active in the background while minimized to the tray.

* **Saved settings**  
  Remembers your selected COM port and baud rate between sessions.

---

* Windows
* .NET Framework 4.7.2 or newer
